Frontend Developer

Cybersecurity · Product Engineering · Bangalore

Mar 2025 — Nov 2025

The product was functional but the frontend was held together by imperative patterns — useEffect-driven API calls, prop drilling, and a monolithic bundle. I took ownership of modernizing the entire architecture.

  • Migrated 90+ components from useEffect to TanStack React Query — eliminating race conditions and introducing server-state caching
  • Designed centralized state with Zustand, replacing scattered sessionStorage and prop-drilling
  • Implemented code splitting with React.lazy across 60+ routes, cutting initial bundle size
  • Built dynamic MUI X Data Grid interfaces with server-side sorting, filtering, and inline actions
  • Introduced animations, collapsible tables, and restructured the entire project architecture
