Sechpoint Pvt Limited Mar 2025 — Nov 2025
Frontend Developer
Cybersecurity · Product Engineering · Bangalore
When I joined Sechpoint, the product was functional but the frontend was held together by imperative patterns — useEffect-driven API calls, prop drilling, no centralized state, and a monolithic bundle. I took ownership of modernizing the entire frontend architecture while continuing to deliver features.
  • Migrated the data layer from scattered useEffect calls to TanStack React Query across 90+ components — introducing server-state caching, background refetching, and eliminating race conditions that plagued the previous approach
  • Designed a centralized state architecture with Zustand, replacing ad-hoc sessionStorage and prop-drilling with typed, predictable stores for authentication flows and device management
  • Implemented route-level code splitting using React.lazy and Suspense for 60+ routes, significantly reducing initial bundle size and time-to-interactive
  • Built advanced data grid interfaces with MUI X Data Grid — server-side sorting, dynamic columns, and inline actions for managing firewall rules, device configs, and subscriber records
  • Introduced UI animation patterns — fade transitions, animated drawer navigation, and collapsible accordion tables — elevating the interface from static to fluid
  • Restructured the project architecture, establishing clear separation between pages, components, hooks, services, and utilities — making onboarding and code reviews significantly smoother
