The Problem

The existing cybersecurity platform had grown organically — imperative patterns everywhere, useEffect-driven API calls causing race conditions, prop drilling across nested components, and a monolithic bundle that took seconds to load. The frontend was functional but architecturally fragile.

  • Race conditions from unmanaged async calls
  • No centralized state — scattered sessionStorage
  • Monolithic bundle — no code splitting
  • No shared component library
auto_fix_high

The Solution

I took ownership of a complete architectural overhaul — migrating to modern React patterns, introducing server-state management, centralizing client state, implementing code splitting, and building a reusable component library that the entire team now ships features on.

  • TanStack Query for server-state caching
  • Zustand stores replacing prop drilling
  • React.lazy across 60+ routes
  • Shared component library (90+ components)
